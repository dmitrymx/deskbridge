# История изменений (Changelog) — DeskBridge

Все важные изменения проекта DeskBridge задокументированы ниже.

## [2.1.0] — 2026-05-24

### Добавлено
- **Горячая клавиша экстренного сброса KVM:** Нажатие `Ctrl + Alt + Escape` на Host-компьютере мгновенно освобождает курсор мыши и клавиатуру, возвращая их на локальный экран.
- **Динамический выбор сетевого адаптера:** В окно настроек добавлен выпадающий список доступных IPv4-интерфейсов (Wi-Fi, Ethernet). Позволяет корректно транслировать IP-адрес соседа, исключая виртуальные интерфейсы WSL/VMware/VirtualBox.
- **Нативный Settings Dialog (HTML5 `<dialog>`):** Окно настроек переведено на стандартный элемент `<dialog>` в соответствии с Chrome Modern Web Guidance для лучшего фокус-треппинга и закрытия по клавише `Esc`.
- **Автозапоминание соседа:** Выбранное устройство теперь сохраняется в `localStorage` и преселектится при перезапуске программы, если оно обнаружено в локальной сети.
- **Индикация скорости передачи:** Появился анимированный виджет скорости передачи файлов (в MB/s) и прогресс-бар в P2P-панели.
- **Справочное руководство разработчика:** Написан подробный русскоязычный `README.md` с инструкцией компиляции для сборки агентами под macOS.

### Исправлено
- **Баг с фиксацией разрешения экрана (KVM Clamping):** Ограничение в `1920x1080` заменено на динамическое определение размера дисплеев хоста и клиента с помощью `rdev::display_size()`. Курсор больше не обрезается на экранах с высоким разрешением (4К и др.).
- **Вечное зависание KVM-клиента:** Добавлен таймаут чтения сокета в 3 секунды на клиенте и отправка пинг-пакетов (Heartbeat) раз в секунду с сервера. При отключении соседа клиент автоматически завершает сессию.
- **Защита от дребезга соединений:** Добавлена блокировка `IS_CONNECTING`, предотвращающая спам-подключения при скольжении курсора по границе.
- **macOS-специфичные пути:** macOS-fallback путь `/Users/a1/Downloads` в загрузчике заменён на кроссплатформенное динамическое разрешение директории загрузок `dirs::home_dir()`.

### Стилизация (Дизайн)
- Полностью обновлена тема оформления под премиум-стандарт: глубокие тёмные тона, стеклянный эффект (Glassmorphism), шрифты *Outfit* и плавные Hover/Active микро-анимации.
- Добавлена индикация статуса KVM ("Управление удаленное" / "Управление локальное").
