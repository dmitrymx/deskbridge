# DeskBridge (v2.2.0) — Локальный KVM-переключатель и P2P-обменник файлами

**DeskBridge** — это кроссплатформенное приложение на базе **Tauri v2, React и Rust**, разработанное для бесшовного взаимодействия между компьютерами (Windows, macOS) в одной локальной сети. 

Приложение не использует облачные сервера, работает напрямую через сокеты и полностью конфиденциально.

---

## 🌟 Основные функции

### 1. Виртуальный KVM (Universal Control)
Управляйте двумя стоящими рядом компьютерами с помощью **одной мыши и клавиатуры**:
*   **Свободный переход границы**: Просто переведите мышь за правый или левый край монитора, и курсор плавно появится на соседнем компьютере, перехватив управление.
*   **Кроссплатформенность**: Вы можете управлять MacBook с Windows-компьютера и наоборот.
*   **Горячие клавиши активации**: Настройте удобную клавиатурную комбинацию переключения режима прямо в панели настроек.
*   **Аварийный сброс**: Если связь оборвалась или мышь застряла на соседнем экране, нажмите **`Ctrl + Alt + Escape`** на управляющем компьютере, чтобы мгновенно вернуть курсор домой.

### 2. Прямой P2P-обмен файлами (High-Speed)
Передавайте файлы любого размера по локальной сети без интернета:
*   Скорость ограничена только возможностями вашего роутера или кабеля (до 1 Гбит/с и выше).
*   Автоматическая проверка целостности файлов по контрольной сумме **SHA-256**.
*   Файлы сохраняются в системную папку **«Загрузки»** (Downloads).

### 3. Мобильный Веб-портал (iOS / Android)
Встроенный веб-сервер для быстрой отправки файлов с мобильных устройств на ПК:
*   В настройках приложения указан локальный адрес (например, `http://192.168.1.15:53203`).
*   Откройте его в браузере вашего iPhone или Android-смартфона (в той же Wi-Fi сети).
*   Загружайте фото и документы без проводов и без установки приложений на телефон.

## 🛠 Что нового в версии v2.2.0

*   **Настраиваемые горячие клавиши KVM**: Добавлена возможность настроить собственную комбинацию модификаторов (Ctrl, Alt, Shift) и клавишу переключения (A-Z, F1-F12) в интерфейсе настроек с мгновенной синхронизацией в Rust-модуль.
*   **Логирование ошибок и крашей**: Добавлена потокобезопасная запись логов приложения в файл `deskbridge.log` с кнопками копирования, очистки и скачивания логов в формате `.txt` прямо в настройках.
*   **100% Offline-Ready (Автономный режим)**: Полностью удалены внешние CDN-зависимости (Google Fonts CDN, иконки, внешние скрипты). Все шрифты переведены на системные, иконки загружаются из бандла `lucide-react`, а мобильный веб-портал полностью автономен.
*   **Устранение краша файлового диалога на Windows**: Вызовы системных диалогов выбора файлов (`pick_file` и `save_file`) перенесены в главный UI-поток приложения через `app_handle.run_on_main_thread`, предотвращая зависание и падение программы на Windows.
*   **Исправление зависания мыши на macOS**: Удален нестабильный вызов `Mouse::get_mouse_position()`, приводивший к сбросу координат в `(0,0)` на macOS. Позиционирование теперь работает на основе относительного смещения (deltas) в памяти.

---

## 🛠 Что нового в версии v2.1.0

*   **Динамическое разрешение**: Использован `rdev::display_size()` для автоматического определения размера экранов. Больше нет необходимости вручную выставлять разрешение в настройках. Курсор корректно масштабируется и не застревает на 4K-дисплеях.
*   **Таймаут сокетов (Heartbeat)**: Сервер отправляет пинг-пакеты каждую секунду, а клиент сбрасывает зависшее соединение через 3 секунды молчания. Никаких «вечных» зависаний при обрыве Wi-Fi.
*   **Выбор адаптера связи**: Добавлена возможность выбрать активный сетевой интерфейс (Wi-Fi, Ethernet) в настройках, чтобы исключить путаницу с виртуальными сетями (WSL, VMware, VirtualBox).
*   **Премиум-дизайн по гайдам Chrome (Modern Web Guidance)**:
    *   Стеклянная тёмная тема (Glassmorphism), шрифты *Outfit* и микро-анимации интерфейса.
    *   Интеграция нативного HTML5-элемента `<dialog>` для окна настроек (автофокус, нативный бэкдроп-блюр, закрытие по клавише `Esc`).
    *   Автозапоминание последнего выбранного устройства в `localStorage`.

## 💻 Инструкция по сборке

### 📋 Предварительные требования (Prerequisites)

#### Для Windows 11:
1.  Установленный [Node.js](https://nodejs.org/) (версии 18+).
2.  Установленный компилятор Rust (`rustup` / `cargo`).
3.  Компилятор C++ (Visual Studio Build Tools с пакетом «Разработка классических приложений на C++»).

#### Для macOS (MacBook):
1.  Установленный [Node.js](https://nodejs.org/) (через Homebrew или официальный сайт).
2.  Установленный Rust:
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
3.  Инструменты командной строки Xcode (Xcode Command Line Tools):
    ```bash
    xcode-select --install
    ```

---

### 🚀 Шаги сборки (Инструкция для разработчиков и AI-агентов)

Выполните следующие команды в корневом каталоге проекта `antigravity_projects`:

1.  **Установка npm-зависимостей:**
    ```bash
    npm install
    ```

2.  **Запуск в режиме разработки (Dev Server + Tauri Window):**
    ```bash
    npm run tauri dev
    ```

3.  **Компиляция финального релиза (Standalone Executable):**
    ```bash
    npm run tauri build
    ```

#### 📦 Результат сборки:
*   **На Windows**: Standalone-файл `.exe` скомпилируется в `src-tauri/target/release/deskbridge.exe`. Также будут созданы `.msi` и `.exe` установщики в папке `target/release/bundle/`.
*   **На macOS**: Сборка создаст готовое приложение `.app` и `.dmg` образ в папке `src-tauri/target/release/bundle/dmg/`.

## 🔑 Разрешения (macOS)
Так как приложение перехватывает ввод мыши и клавиатуры, на macOS необходимо предоставить права доступа:
1.  Перейдите в **Системные настройки > Конфиденциальность и безопасность > Универсальный доступ** (Accessibility).
2.  Добавьте **DeskBridge** в список разрешённых приложений и включите тумблер.
